# What is VitePress?

VitePress is a [Static Site Generator](https://en.wikipedia.org/wiki/Static_site_generator) (SSG)
designed for building fast, content-centric websites.

```bash
npm add -D vitepress
npx vitepress init
```

Read the guide for more.
