Introduction

Nest (NestJS) is a framework for building efficient, scalable Node.js server-side applications. It uses progressive JavaScript, is built with and fully supports TypeScript, and combines elements of object-oriented programming, functional programming, and functional reactive programming.

Under the hood, Nest makes use of robust HTTP server frameworks like Express (the default) and can optionally be configured to use Fastify as well. Nest provides a level of abstraction above these common Node.js frameworks, but also exposes their APIs directly to the developer.

This allows developers the freedom to use the myriad of third-party modules which are available for the underlying platform.
