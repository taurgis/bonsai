# Migration of Active Data in B2C Commerce

Publish Date: Nov 25, 2025

Description

Active Data from environments other than B2C Commerce cannot be imported into your B2C Commerce Production instance.

Resolution

If you are migrating Active Data from sites on the same realm, you can follow the steps in [Manually Migrate Active Data to a New Site](https://help.salesforce.com/s/articleView?id=cc.b2c_manually_migrate_active_data.htm&type=5&language=en_US).

If you would like to migrate Active Data from a site on one B2C Commerce realm to a site on another B2C Commerce realm, please reach out to Commerce Cloud Support for further assistance as outlined in [How to engage Commerce Cloud Support via the Salesforce Help portal](https://help.salesforce.com/s/articleView?type=1&id=Updated-experience-when-engaging-Commerce-Cloud-Support-in-the-Salesforce-Help-portal&language=en_US). Please note that the migration will take a few days so please reach out at least **ten days** before you put your legacy site into Maintenance Status, and before the Go-Live of your new site.
