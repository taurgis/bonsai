# File References

Use relative paths from the skill root when linking to other files.

## Examples

- [Detailed reference](references/REFERENCE.md)
- [Template file](assets/TEMPLATE.md)
- [Script entrypoint](scripts/run.sh)

## Rules

- Keep references one level deep from SKILL.md.
- Avoid chaining references (do not link from one reference file to another).
- Prefer short, focused reference files to reduce context usage.
